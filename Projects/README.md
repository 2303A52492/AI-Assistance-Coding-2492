# 🚀 DocuGenius AI

### AI-Powered Code Documentation Generator

> Paste your code, choose a format, and let AI generate beautiful documentation instantly.

---

## ✨ Features

- 🤖 **AI-Powered** — Uses Google Gemini AI for intelligent documentation generation
- 📝 **3 Output Formats** — Inline comments, Markdown docs, or full README generation
- 🌐 **Multi-Language Support** — Python, JavaScript, Java, C++, Go, Rust, and more
- 🎨 **Premium Dark UI** — Stunning glassmorphism design with smooth animations
- 📋 **Copy & Download** — One-click copy or download generated docs as files
- ⌨️ **Keyboard Shortcuts** — Ctrl+Enter to generate instantly
- 📱 **Responsive Design** — Works on desktop, tablet, and mobile

---

| Layer | Technology |

## 📦 Installation

### Prerequisites
- [Node.js](https://nodejs.org/) (v18 or higher)
- [Google Gemini API Key](https://aistudio.google.com) (free)

### Steps

1. **Clone the repository**
   ```bash
   git clone <your-repo-url>
   cd AI-ASSISTED-CODING-PROJECT
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Configure API key**
   - Open `.env` and replace `your_api_key_here` with your Gemini API key
   - OR enter it through the UI (click the 🔑 API Key button)

4. **Start the server**
   ```bash
   npm start
   ```

5. **Open in browser**
   ```
   http://localhost:3000
   ```

---

## 🎯 Usage

1. **Paste your code** in the left panel
2. **Select the language** (or leave on Auto Detect)
3. **Choose output format:**
   - **Markdown** — Structured documentation with tables and examples
   - **Inline** — Comments added directly into your code
   - **README** — Full project README.md file
4. **Click Generate Docs** (or press `Ctrl+Enter`)
5. **Copy or Download** the generated documentation
